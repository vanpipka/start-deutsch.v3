Политика в отношении обработки персональных данных

Редакция от 30 апреля 2026 г.

1. Общие положения

Настоящая политика обработки персональных данных составлена в соответствии с требованиями Федерального закона от 27.07.2006. №152-ФЗ «О персональных данных» и определяет порядок обработки персональных данных и меры по обеспечению безопасности персональных данных, предпринимаемые владельцем сайта https://start-deutsch.ru (далее — Оператор).

2. Состав собираемых данных

Оператор может обрабатывать следующие персональные данные Пользователя:

  • Адрес электронной почты (e-mail);
  • Имя пользователя (логин);
  • Обезличенные данные о посетителях (в т.ч. файлы «cookie») с помощью сервисов интернет-статистики.

3. Цели обработки

Цель обработки персональных данных Пользователя — предоставление доступа к учебным материалам, тестам уровней A1-B2, сохранение прогресса обучения и направление уведомлений о работе сервиса.

4. Правовые основания

Оператор обрабатывает персональные данные Пользователя только в случае их самостоятельного заполнения Пользователем через формы регистрации на Сайте. Нажимая кнопку «Регистрация» или аналогичную, Пользователь выражает свое согласие с данной Политикой.

5. Хранение и безопасность

Безопасность персональных данных обеспечивается путем реализации правовых, организационных и технических мер. Обработка и хранение данных осуществляются на серверах, расположенных на территории Российской Федерации.

6. Контакты

Пользователь может получить любые разъяснения по вопросам обработки своих персональных данных, обратившись к Оператору через электронную почту: {{company_email}}

7. Использование сервисов аналитики (Cookies)

Для улучшения работы Сайта Оператор использует сервис веб-аналитики Яндекс.Метрика (предоставляется ООО «ЯНДЕКС», 119021, Россия, Москва, ул. Л. Толстого, 16).

Данный сервис собирает обезличенные данные о посещении сайта (IP-адрес, тип браузера, время нахождения на сайте и др.) для анализа пользовательской активности. Настройка сбора данных производится Пользователем самостоятельно в настройках своего браузера. Оператор не использует данные сервиса для идентификации личности Пользователя.

8. Место обработки данных

Оператор осуществляет обработку и хранение персональных данных Пользователей исключительно на серверах, физически расположенных на территории Российской Федерации, в соответствии с требованиями ч. 5 ст. 18 Закона № 152-ФЗ.
